POSTERS FOLDER
===============

Place your event poster images here!

Supported formats:
- JPG/JPEG (best for photos)
- PNG (best for graphics with transparency)
- WebP (modern, smaller file size)
- SVG (vector graphics)

Recommended image size:
- Width: 800-1200px
- Height: 1200-1600px (portrait/poster ratio)
- OR landscape: 1200x800px
- File size: Under 500KB for fast loading

How to use:
1. Place your image file here (e.g., "my-event.jpg")
2. In src/data/events.js, reference it as:
   poster: '/posters/my-event.jpg'

Example:
If you have: public/posters/workshop.jpg
Use in code: poster: '/posters/workshop.jpg'

See IMAGE_GUIDE.txt in the root folder for more detailed instructions!
